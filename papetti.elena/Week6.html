To make the board I use Eagle freeware version to redesign the hello board. All components that I need are in the Library to add to the Eagle basic one. I downloaded it from PROVIDENCE TUTORIAL.
I started studying basic tutorials of Eagle following mainly the process as the video and reading the Data Sheet of each component that interested me. After loading the library and opening the Hello World board I started to re-design the bord.
The interface of the software is quite simple. We are adding a lot of nice components like a button that will controll the connection of the LED with the VCC. I placed all the extra-components over the schematics and I assigned each net-structure to the pins of the microcontroller, to the GNR and to the VCC.
To do the board the next step is again use the Roland Modela in order to mill the board. So first thing to do is export from Eagle PNG the image of the circuit chosing monochrome with a resolution of minimum 600px. Then in photoshop generate the two files for the traces and interior.
With the Roland machine I repeat a procedere already known : the tool 1/64 for traces and 1/32 for interior.
Once all is ready start to put the components. Once the components are there check the connection with a multimeter and ready to play with the ISP to run the bootloader.











COMPONENTS USED:
- 2 Resistor 10 K
- 1 Red Led 1.8 V
- 1 Resistor 180 Ohm
- Attiny 44 microcontroller
- 1 Capacitor Uf
- 1 Cer Resonator 20 MHz
- Black Button Tactile Switch
- Isp
- FTDI
