Week 7

Embedded Programming

Programming the Pushbutton + LED Board

Assignment: HOW TO PROGRAM MY BOARD.
- The aim of this assignment is to program the board we designed. It is a Hello World Board with an extra pushbutton and a LED. It was a real challenge for me to make this work.

THE PROCESS

I followed the tutorials http://academy.cba.mit.edu/content/tutorials/akf/embedded_programming_arduinoIDE.html

- 1 Download the Arduino 1.0 software
- 2 Download and install the ATtiny Board Files
- 3 Make sure I had the necessary drivers installed (ftdi)In the board view distribute the components in that way that they create easy connections. Draw the routes in my board i have used one button and one LED.

I installed the Arduino 1.0 software for Mac, as well as the attiny44 Board files and the drivers. However, when I connected my FabISP and Hello World Board to my computer, it was not recognized.


I tried for the second time.
- I successfully installed Arduino 1.0 and the board files for Mac, but the FabISP was recognized as an "Unknown Device" . When I looked at the device properties, I got a message saying "No drivers are installed for this device" so I went and tried to installed them. It didn't work!.

I tried and tried, but the result is not changed. I want to try to replace some part of my board, although I know it is a desperate attempt because the tester is positive for all tracks
-All for now, over and out.

I tried a third time.
- The third attempt was the good one. The USB port on the ISP board came off and I had to make it again. In this way, I was able to program the Hello board. I made it, folks!
