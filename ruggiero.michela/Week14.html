Interface-Output Devices

Interface and Application Programming

Assignment: write an application that interfaces with an input &/or output device.

I chose to use my output: SERVO MOTOR

For this assignment my goal was to visualize the rotation of the servo motor. I used my output project, and I am going to use it in my final project.

Firstly I downloaded Python, from the webpage Python.org. I don't have much experience with processing, so i made something simple. I made a rectangular root interface in processing, divided by Cartesian axes and the components were positioned according to the Cartesian coordinates. Left and right inputs rotate according to the rotation angle of a servo motor that I am controling through FabMiky. The first step for programming the interface, was to use an Arduino code in my board that would send information to Processing.The degree (angle) location of the servo motor are the data that FabMiky send to the personal computer and Python reads through the serial port.
