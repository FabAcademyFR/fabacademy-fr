Week 14

Interface-Output Devices

Interface and Application Programming

Assignment: write an application that interfaces with an input &/or output device.

interface_application_programming/index.html

I chose to use my output: SERVO MOTOR

To develop my interface I decided to use Python, because FabLab Frosinone during this week organized a class about it. I take this opportunity to thank them.

Python is a very powerfull programming language. It has a very simple grammar, so is very intuitive to use to develop even complex application. For this exercise I attend a python class in wich i learnt how to use pyserial to comunicate via serial port to the fabkit, and Tkinker to build a gui. Design a graphical user interface with Tkinker is an easy job: every element (labels, buttons, etc.) is a widget, and it is positioned in the main window according to Cartesian coordinates.

For this assignment my goal was to visualize the rotation of the servo motor. I used my output project for the hardware part, as I am going to use it in my final project.

Firstly I downloaded Python, from the webpage Python.org. I do not have much experience with this language, so I made a basic gui application wich have two bottuns (left and right). Clicking on the buttons results in a certain number to be sent via serial to the fabkit (thanks to the FTDI adapter). The Arduino code on the FabMiky read the number and move the servo motor accordingly.

The degree (angle) location of the servo motor are sent from FabMiky to the computer, so Python can read them through the serial port.
